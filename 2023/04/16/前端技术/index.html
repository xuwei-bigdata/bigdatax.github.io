3.1 npm

3.1.1 npm 相关命令

命令 作用 备注
npm init （-y） 初始化npm项目 npm init jquery@2.1.x 下载jquery并指定版本
npm install 依赖名 下载依赖
npm config get registry 查看使用的依赖仓库
npm config set registry https://registry.npm.taobao.org 配置淘宝的npm镜像
npm config list 查看配置信息
npm update 包名 更新包
npm update -g 包名 全局更新
npm uninstall （-g） 包名 卸载包
